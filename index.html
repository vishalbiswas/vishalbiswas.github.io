CodeNameMSIC

CodeNameMSIC is a music player written in C#. NAudio forms the base of the actual playack controls. Its based on .Net 4.0.

Splash

Splash was supposed to be a news app, which later turned into a forum like application which would allow for custom chat servers. Essentially, you could type the address of a web service and it will use that server for communication, kinda like the IRC chat clients. Except, Splash would allow for posts and thread oriented communication and user rights management instead of plain text chatting.

ChessWars-unity

ChessWars is my latest project. It hosts all the CSharp scripts that will be used in my upcoming game Chess Wars, based on the Unity game engine. The scripts are supposed to be cross engine with as little overhead as possible.
